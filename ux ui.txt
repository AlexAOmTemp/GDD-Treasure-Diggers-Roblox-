ux ui.txt — Обновлённая версия
1. HUD & Основные элементы

1.1 Coins & AFK / Friends Bonus

Отображается текущее количество монет игрока.

AFK Income и Friends Bonus показываются отдельными иконками с подсказками:

AFK: “50% Base Income (может быть увеличено до 100% за счёт Rebirth Shop)” — **только в AFK-зоне**

Friends: “+10% per friend, max 100%”

Расчёт дохода полностью на сервере → предотвращает читерство.

1.2 Inventory / Pets Slots

Максимум 50 слотов для питомцев + улучшения из Rebirth Shop (petSlots).

**Equip Slots: базовое количество + улучшения из Rebirth Shop (equipSlots) + Game Pass.**

Bulk Open (3 или 8 яиц) → большие области нажатия для мобильных.

Equipped Pets выделяются рамкой + индикатор XP.

При переполнении инвентаря появляется предупреждение и кнопка «Manage Inventory».

1.3 Pet XP & Level

XP Bar для каждого питомца на HUD (Mobile Safe Zone).

Прогресс учитывает клики, AFK Income, скармливание и Fusion.

После Level Up: всплывающее VFX + звуковой эффект.

Уровень питомца влияет на множитель дохода (см. economy.txt).

1.4 Bulk Open Feedback

Анимация открытия яиц для 3/8 единиц сразу. Анимация показывает каждое открытие, но результат отображается после обработки всех.

Если транзакция не удалась (ошибка DataStore или откат), показывается всплывающее уведомление “Transaction Failed, No Items Lost”. **На время обработки запроса Bulk Open интерфейс блокируется (кнопки неактивны, отображается индикатор загрузки).**

2. Rebirth / Rebirth Shop

2.1 Rebirth Screen

Кнопка [Rebirth] с подтверждением и подсказкой “Gain Rebirth Points & reset progress”. **Кулдаун между Rebirth (отображается серверное ограничение).**

Полноэкранный VFX + звуковой эффект при завершении Rebirth.

2.2 Rebirth Shop

Покупка перманентных бонусов за очки Rebirth:

BonusIncome +%

ClickRateLimit увеличение

Slot Expansion (увеличение инвентаря)

**Equip Slot Expansion (дополнительные слоты экипировки)**

AFK Income +%

Chest Spawn Rate +%

Кнопка «Spend on Bonuses» → открывает список доступных бонусов.

Цены в очках Rebirth и эффекты бонусов указаны прямо на кнопках.

3. Egg Hatch / Bulk Open UI

Возможность открыть 1 / 3 / 8 яиц сразу.

Каждое яйцо показывает прогресс открытия + анимацию выпадения питомца.

Pity System визуализирована: индикатор растущего шанса Legendary при неудачных попытках. Индикатор обновляется после каждого открытия.

После открытия отображается краткая инфо-карта питомца:

Type, Level, Multiplier, FusionStatus, Golden.

4. Quests & Notifications

Quest Tracker на HUD (Mobile Safe Zone).

Поддержка Daily / Event / Mission Quests.

После выполнения: всплывающее окно + монеты / яйца / очки Rebirth.

Уведомления анимированы + звуковой сигнал.

При изменении списка квестов (новый день) обновляется весь трекер.

**Уведомления об ошибках:**
Все уведомления об ошибках отображаются в единой области HUD с приоритетом по времени. Одновременно может отображаться только одно уведомление, остальные ставятся в очередь.

5. Mega Chest & Floating Feedback

Mega Chest HUD: индикатор HP + таймер спавна. HP обновляется не чаще 0.2 сек.

Floating Numbers для монет и урона отображаются на серверных событиях (Object Pool). Не более 10 одновременно.

VFX адаптировано под мобильные устройства → минимизация лагов.

6. Mobile Adaptation

Все кнопки ≥ 80px, включая Bulk Open и Rebirth Shop.

Safe Zone учтена для устройств с челками / Notch.

Доступ к инвентарю и меню Rebirth через нижнюю панель.

Анимации и эффекты масштабируются для разрешений от 720p до 4K. Использовать UIScale.

7. Security & Feedback Loop

Все важные действия (Rebirth, Bulk Open, Chest Destroy) проверяются на сервере.

Ошибки транзакций отображаются на HUD с подсказкой «Retry».

Feedback Loop: каждый крупный прогресс (Level Up, Rebirth, Golden Pet) сопровождается VFX + звуковым сигналом для мотивации игрока.

Дополнительно: на клиенте установить локальный RateLimit для предотвращения спама кнопок.

**Приоритет VFX при множественных событиях:**
При одновременном наступлении нескольких событий (например, Level Up + Hatch) приоритет отдаётся событиям с более высокой значимостью: Rebirth > Golden Pet > Level Up > Hatch > Fusion. VFX воспроизводятся последовательно, но допустимо наложение кратких эффектов.
